# Squad HERMES — Contexto Canônico

Este documento é a fonte canônica de contexto para a Squad HERMES. Ele é carregado no início de toda sessão por todas as IDEs.

**O que é a HERMES?**
Hierarchical Engineering Reverse-Map & Extraction Squad (HERMES) é uma equipe agentic projetada para fazer engenharia reversa de artefatos de software e gerar Software Design Documents (SDD).

**Regra de Ouro:**
Nenhuma alteração é feita no código original do usuário. Toda saída é gravada em `_hermes/{scope-slug}/`. Todos os agentes seguem as diretrizes especificadas em `_codesteer/`.
